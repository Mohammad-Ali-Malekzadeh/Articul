    4. grid: "a" minmax(100px, max-content) "b" 20%;
      "a" Area
      "b" Area

      The minmax() CSS function defines a size range greater than or equal to min and less than or equal to max. It is used with CSS Grids.

    5. grid: 100px / 200px;

      When we use a slash between grid values, it means to set the row and height.

      grid rows / grid columns.

      In this example, only the first row is set to 100px.

    6. grid: minmax(400px, min-content) / repeat(auto-fill, 50px);

      In the example above, the repeat() function lets us specify how many columns we want (auto-fill), and what size they should all be (50px). (From sitepoint.com)

      auto-fill allow for varying numbers of tracks depending on the available space. This makes them very handy for responsive layouts without media queries. (From sitepoint.com)

      If the min value in minmax(min, max) is larger than the max, then the max will be ignored. The minmax(min, max) will be treated as a min value.

      in this example when the screen width is low, we'll see that only the first row obeys minmax(). if we want minmax() size to apply to total rows, we need to insert minmax function in repeat(). For Example: repeat(auto-fill, minmax(200px, 1fr)). (From ishadeed.com)
